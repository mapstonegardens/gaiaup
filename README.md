# gaiaup

![gaia banner](https://drive.google.com/uc?export=view&id=171_LwWqx4zUGKftCVtWlN-zp0NgqTZ3m)


# Installation
Install using pip
```python

```

```python
from gaiaup import seismic2d, seismic3d, wavefield3d
```
